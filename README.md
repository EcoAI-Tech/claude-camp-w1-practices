# claude-camp-w1-practices

Week 1 practice exercises for Claude Camp. This repository contains 5 small Python scripts (each under 30 lines) built with the help of Claude Pro. The main goal of this week is **not** to write the most elegant code, but to walk through the full workflow: writing code → committing with Git → pushing to GitHub.

## 📋 Exercises

| # | Script | Description |
|---|--------|-------------|
| 1 | `temperature_converter.py` | Convert between Celsius and Fahrenheit |
| 2 | `personalized_greeting.py` | Take a name and age as input, output a formatted greeting |
| 3 | `tip_calculator.py` | Take a bill amount and tip percentage, output the total |
| 4 | `password_generator.py` | Take a length from the user, generate a random password |
| 5 | `bmi_calculator.py` | Take height and weight, output BMI and a health suggestion |

## 🛠️ Tech Stack

- **Language:** Python 3
- **AI Assistant:** Claude Pro
- **Editor:** VS Code
- **Version Control:** Git + GitHub

## 🚀 How to Run

Make sure Python 3 is installed, then run any script from the terminal:

```bash
python temperature_converter.py
python personalized_greeting.py
python tip_calculator.py
python password_generator.py
python bmi_calculator.py
```

## 📝 Workflow

This week's recommended workflow:

1. Create a GitHub repository named `claude-camp-w1-练习`
2. Clone it locally with `git clone`
3. Write the 5 `.py` files in VS Code
4. Use Claude Pro to assist with the code (and keep screenshots of the conversations)
5. Commit and push the changes:
   ```bash
   git add .
   git commit -m "Add week 1 practice scripts"
   git push
   ```
6. Share the repository link in the check-in group
